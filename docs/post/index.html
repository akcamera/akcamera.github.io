撮影に至るまでの情報整理

私の場合は基礎的な人物撮影の知見がある状態から技術の枝を伸ばす目的でコスプレ撮影の世界に入りました。 ライティング含む人物撮影における技術の蓄積がある状態で活動をはじめ、アウトプットの場としてコスプレ撮影を選択しました。（身内が始めて楽しそうというのも大きな理由の一つです) 特殊撮影など活動開始初期から自分がやりたいと思った撮影を概ね初期から実現させてもらったので自分のペースで技術を習得することができました。 この界隈では誰を撮っているかが重要視されます。これについて個人的にはとてもうんざりするのですが、全体の価値観として根付いているので致し方ないところでしょう。 基本的にはそういう事に関係なく仲良くしてくれるレイヤーさんをいい感じに撮って活動できれば満足という感じでやっています。 長く撮影をしているとすべて自分のコントロールで写真を撮るわけではないというのに気が付くと思います。 撮影毎に役割や求められるケースが違っており、これを整理しようというのが記事のテーマです。 撮影のリーダーが誰なのかは割と重要です。 もちろん双方というケースもありますが、撮影にかかる熱意はどちらかに傾いているというのが実情ではないでしょうか。 カメラマンとしてはそのレイヤーを撮りたいというのと、レイヤーとしてはこの衣装を撮りたいというのが満たされるケースが満足度の高い撮影に繋がるのかなと個人的には思ってます。 あの人の、あのキャラを、自分の作品として撮りたい！という人もいれば、撮影行為そのものや撮影後の交流に価値を感じるカメラマンもいらっしゃいます。 撮影毎に被写体もカメラマンも目的が違うといそれはまた別の話として、アウトプットイメージについて誰がどのように決めるのかという話です。 ここではミスマッチが起こり勝ちだなーと感じるケースや撮影の満足度が高かったケースを掘り下げます。 以下は誰が主体で各項目を決めるのかというマトリクスです。 👫=レイヤー 📸=カメラマン パターン 衣装 ロケ(スタジオ) 内容(テーマ) A 👫 👫 👫 B 📸 📸 📸 C 👫/📸 👫/📸 👫/📸 D 👫 👫 📸 E 📸 📸 👫 被写体からの撮影依頼やカメラマン募集案件 僕の所感ですが、レイヤーさんがイメージリーダーの場合 衣装と内容をレイヤーさんが決める(A) 場所や日時についてはカメラマンと内容の合意のあと決める(C) このケースは割と撮影しやすいなと思っています。 例えば暗いイメージの写真を、白ホリで撮れるケースは相応の技術にライティングや構図を限定する必要があります。 顔が盛れてキャラの影は少ないけど背景は落とす、しかも全身、というような撮影をしたいとなったら奥行きが必要になります。 光を切るためのスペースも相応に必要です。 上記は一例ですが、カメラマンがしっかりしていれば取捨選択と提案できるアイデアを持っているはず。 場所や日時のすり合わせはカメラマンとすり合わせを行うといい結果を生みやすくなります。 検討している場所を伝えてカメラマンがイメージに近い撮影ができるフィールドで予定を決める事がお互いの満足感にも繋がるかな。 自分の場合、撮影できないケースはないというかネガティブをポジティブに変える術に長けている自信はあります。 でも、できれば無理せずストレートに撮影したいというのが本音です。 なので被写体さんにイメージがあるケースであればカメラマンが力量や物理的な想定をして撮影環境を選べるケースが双方納得感が出やすいのかなと思ってます。 Ａのケースでは割とカメラマンの満足感は低めになり勝ちです。 カメラマンはすべての情報が開示されているので、イメージに合わない人は先に振り落とされるはずですが、募集などの場合はここから男装の撮影に対する募集なのに 女装をねじ込んできたり内容を曲げようとする人がいるのでこういう人は断りましょう笑 撮影してほしいという弱みにつけ込むタイプの人です。 逆に衣装も場所もレイヤーさんが決めているが、カメラマンに内容を求めるケース(D) これはカメラマンがそれなりに撮れるタイプだとありがちなケースですが、初回くらいにしましょう。 個人的には一番しんどいケースです。仕事で汎用的な撮り方をするのは全然ありなのですが。 自分の場合はジャンルを縛っておくことでこの辺を回避しています。 結局いつもと同じような撮影になると飽きてしまって満足度は低くなりますね…。 カメラマンが衣装や被写体を指名した撮影 カメラマン的にはアガる撮影になりやすいケースですね笑 対してレイヤーさんの方は受け身になりがちなので温度差が難しい。 カメラマンが衣装を決めている場合は内容もカメラマンが決めるのが自然な流れなのですが、たまにこれをレイヤーさんに投げる人がいます。 Eのケース、つまりDの逆パターンですね。 被写体側もその撮影で撮りたいイメージを持っていればお互い保完できるので満足感があるかな。 カメラマンからのアプローチで撮影が決まり、この二つを指定したにも関わらずテーマなどの提示がなく被写体任せな撮影になるケースは被写体側からは不満が出る結果になりがちです。...
